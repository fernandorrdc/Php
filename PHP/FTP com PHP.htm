FTP com PHP

Olá , esse é meu primeiro artigo aqui na Revista PHP.

Objetivo desta Matéria: Utilizar algumas funções de FTP no PHP e por fim fazer transferência de arquivo

Bom pessoal, vamos ao código 

<?php
// definindo o tempo limite da aplicação o time default é de 30 segundos 
// você setando para 0  fica um tempo indefinido
