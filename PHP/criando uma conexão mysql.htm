Criando uma conexão Mysql

Olá, pessoal.
Vocês ao navegarem pela internet já devem ter vistos diversas maneiras de como criar uma conexão mysql, mais de todas que você vê cada uma diferente hoje irei ensinar para vocês o jeito mais “fácil” de como criar uma conexão com as funções básicas para vocês entenderem cada função delas, assim como em outros tutoriais irei mostrar como fazer uma seleção do banco de dados exibir esses dados entre outros, resolvi fazer esse tutorial em escrita em vez de aula para assim vocês prestarem mais atenção em cada função, pois esse misto de PHP e MySQL é muito ótimo para desenvolvimentos de sites WEB. Bom vamos para de tanta “conversa” e ir logo para o tutorial.

1º Como vocês podem ver abaixo na imagem uma conexão MySQL já programada.


2º $conexao, como vocês já sabem isso se chama uma variável que é introduzida um dólar para identificar o mesmo, seguido do nome da nossa variável que no nosso caso conexao, conexao usei como exemplo, mais vocês devem saber que ela não é uma função e sim uma string de identificação que executa nossas funções.

3º mysql_connect, está é sim uma função que estabele uma conexao com o nosso servidor MySQL, podemos dizer que em português essa função quer dizer “Conexao com o MySQL” logicamente.

4º (“Endereço do meu servidor”), como vocês observam na imagem acima devemos colocar nosso endereço do servidor MySQL, que normalmente se não maioria dos servidores são localhost, caso não seja localhost normalmente é um IP, ali colocaremos o endereço do nosso servidor que no nosso exemplo abaixo é localhost.


5º Logo apôs colocaremos nosso usuário da nossa conexão MySQL, no meu caso é root, como mostra a imagem abaixo.


6º Como vocês podem ver abaixo colocamos nossa senha MySQL, que no meu caso foi 123


7º e por último vejamos a função or die, está função nos surgira uma outra função, caso aconteça algum problema com nossa conexao.


8º Apôs está função colocaremos entre parênteses a função mysql_error que se caso aconteça algum erro com nossa conexão na hora de se conectar ira surgir um erro na tela mostrando qual erro aconteceu seja na função no servidor usuário e outro.


9º Mais caso você queira também em vês de mysql_error podemos por uma mensagem como por exemplo “Não foi possível conectar”, como você pode ver abaixo.


10º E por ùltimo para você tiver certeza de que tudo ocorreu bem podemos dar um echo na tela dizendo a seguinte mensagem “Conexão feita com sucesso”.


Bom é isso espero que tenham gostado como você pode ver foi explicado o máximo possível para assim iniciantes nessa linguagem entenderem um pouco melhor.
